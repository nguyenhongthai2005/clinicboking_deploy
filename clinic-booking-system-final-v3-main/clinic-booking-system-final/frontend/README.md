Hướng dẫn pull dự án phần frontend.
- Sau khi pull dự án về:
+ Open Terminal
+ Gõ lệnh "cd clinic-booking-system-final/frontend" để đến thư mục frontend.
+ Gõ lệnh "npm i axios bootstrap react-bootstrap react-router-dom react-router-bootstrap" để import các thư viện cần thiết để chạy dự án.
+ Gõ lệnh "npm i dotenv" để import thư viện dotenv.
+ Gõ lệnh "npm i jwt-decode" để import thư viện jwt.
+ Từ thư mục frontend, New File và rename ".env" và liên hệ vophammy@gmail.com để lấy được nội dung file.
+ Trong file .gitignore thêm nội dung ".env" vào cuối file để không upload file này lên github (vì lý do bảo mật).
+ Để chạy dự án: cd đến frontend, gõ lệnh "npm run dev".
